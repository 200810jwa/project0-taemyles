# Banking Application

CLI banking application for customers and bank employees.
<br>
## Functionalities
**Customers**
- Register
- Apply to open an account
- Withdraw
- Deposit
- Transfer funds

**Employees**
- Account info
- Account balances
- Personal info
- Approve/deny applications

**Admins**
- Inherit employees functionalities
- Withdrawl, deposit, transfer from all accounts
- Cancel accounts
